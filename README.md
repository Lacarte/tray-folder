



# tray-folder

  
### To set venv to the project

    python -m venv tray-folder\venv

  
### To activate venv 

    venv\Scripts\activate.bat

  

### To desactivate venv

    venv\Scripts\desactivate.bat

  
### To backup libraries

    pip freeze > requirements.txt

  
### To install libraries

    pip install -r requirements.txt
